# Service Analytical Platform

A modern, vanilla JS analytical dashboard with a Node.js backend.

## How to Run

### 1. Start the Backend
The backend handles authentication and provides data for the charts and tables.

```bash
cd backend
node server.js
```
The server will start at `http://localhost:5000`.

### 2. Open the Frontend
Since there are no build tools, you can open the frontend directly in your browser.

- Locate and double-click `frontend/index.html`
- Or use the absolute path: `file:///C:/Users/subas/OneDrive/Desktop/analytics/frontend/index.html`
